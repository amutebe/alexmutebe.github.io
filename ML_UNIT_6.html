Clustering with Python

In this Unit, I learned various data science techniques, including data cleaning, exploratory data analysis (EDA), and hypothesis testing. Specifically, for this assignment, I focused on analyzing the relationship between Airbnb listing prices and their neighborhoods. Using tools like pandas for data manipulation and seaborn for visualization, I was able to uncover trends and patterns in the data.

The skills acquired in this module are crucial for conducting meaningful business analyses. Understanding the price dynamics across different neighborhoods can provide valuable insights for both Airbnb hosts and guests. Hosts can adjust their pricing strategies to remain competitive, while guests can find the best value for their money. Moreover, these data science techniques are broadly applicable in various industries, highlighting the importance of data-driven decision-making.

Moving forward, I plan to deepen my knowledge of advanced statistical methods and machine learning techniques to further enhance my analytical capabilities. Additionally, I aim to apply these skills in real-world projects, potentially exploring other factors influencing Airbnb prices, such as property type or host experience. The insights gained from this analysis could also inform Airbnb's marketing and pricing strategies, ultimately leading to better service for users.

Task A: Iris data

We performed K-Means clustering on the dataset, iris.csv (from the UCI Machine Learning Repository). Before using the data for clustering. We chose various values of K;Using the elbow method, K = 3 had best results.

Observations: With K=3 Setosa classification is at 100% accurate rate Versicolor classification is at 100% accuracy rate Virginica has lowest classification at 64% accuracy rate This means that the clustering is very accurate for setosa and versicolor, but less accurate for virginica. To quantify this, we can calculate the accuracy for each species: Setosa: 100% (50/50) Versicolor: 100% (50/50) Virginica: 64% (32/50) Overall, the clustering is quite good, with an average accuracy of 88%.

Task B: Wine data

We performed K-Means clustering on the dataset, wine.csv (from the UCI Machine Learning Repository). Before using the data for clustering, we performed Exploratory Analysis to remove unneccessary data.

Observations: With K=3 Setosa classification is at 100% accurate rate Versicolor only 2 were misclassified 96% accuracy rate Virginica has highest misclassification at 72% accuracy rate This means that the clustering is very accurate for setosa and versicolor, but less accurate for virginica. To quantify this, we can calculate the accuracy for each species: Setosa: 100% (50/50) Versicolor: 96% (48/50) Virginica: 72% (36/50) Overall, the clustering is quite good, with an average accuracy of 89%.
